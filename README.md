# Password Generator 

This is a Python script that allows users to create accounts by entering their name, phone number, email, and password. It includes:
- Email validation (`@gmail.com`)
- Strong password checks
- Option to auto-generate a secure password
- Clean user prompts and final account summary

##Sample Output

Your account is created successfully. :)

Your contact: 03001234567
